## Installation

```bash
cd [workspace]/src
git clone https://github.com/Juunghyeon/pedsim_motion.git
cd ./build
cmake ..
make
export GAZEBO_PLUGIN_PATH=${GAZEBO_PLUGIN_PATH}:[workspace]/build
cd ../worlds
gazebo --verbose Cafe.world
```
